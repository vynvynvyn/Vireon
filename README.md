# Vireon
Universal Typewriter Bot

## How it works
- You take a snapshot using a keybind of your choice, of a region of your screen containing text.
- The code uses **Tesseract**, an open model to recognize and identify this text.
- You press a specific keybind for writing this recognized text onto a textbox.  (All alphabets are induvidually written to prevent detection)

## Requirements
- Python
- Several python modules are required for this model, the starter batch file already installs these
- Tesseract (an open-source engine that reads text from images or screenshots and converts it into editable, machine-readable text)]

## Is it safe?
- Yes this model is completely safe and opensource.
